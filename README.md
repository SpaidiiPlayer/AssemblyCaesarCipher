  <li>Descriptografia
 
<h4>Manual para o usuário: <h4\>
 <ol>

  <li>É solicita o nome dos arquivos de entrada e saida.
  <li>É criado\modificador o arquivo de saída com o texto criptografado ou descriptografado.
